# Technical Foundation

## Platform

- Unreal Engine 5
- Third-person multiplayer
- Dedicated server preferred
- Server-authoritative outcomes for combat, inventory, evidence, voting, and faction state

## Gameplay Ability System (GAS)

Suggested attributes: wealth, public support, noble support, church support, military strength, information access, fear, reputation, suspicion, and trust.

Suggested tags:

- `State.Suspected`
- `State.Imprisoned`
- `State.Exiled`
- `State.Protected`
- `Action.Assassination`
- `Action.Investigation`
- `Council.VoteOpen`

## Data-driven design

Use data tables or primary data assets for action costs, faction reactions, evidence rules, NPC archetypes, and event parameters. Designers must be able to tune balance without code changes.
